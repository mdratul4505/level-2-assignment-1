# Advanced Problem Solving with TypeScript & OOP

## Overview
This assignment demonstrates fundamental TypeScript concepts including data typing,
interfaces, class inheritance, type checking, and data structure manipulation.

## File Structure
```
├── solutions.ts   # All 7 problem solutions
├── blog-1.md      # Blog: any vs unknown & type narrowing
├── blog-2.md      # Blog: Pick and Omit utility types
└── README.md      # Project overview
```

## Problems Covered
1. Filter Even Numbers
2. Reverse String
3. Union Type & Type Guards
4. Generic Function with Constraints
5. Interface with Utility Function
6. Class Inheritance (Person → Student)
7. Array Intersection

## How to Run
```bash
node solutions.ts
```
